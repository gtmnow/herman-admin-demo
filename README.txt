How to use
1. Open index.html in a local web browser.
2. Use the left navigation to switch between the current-state admin tool surfaces.
3. The demo is non-functional and intended for visual review only.

- Portal branding setup with custom logo URL and welcome message.
- Current settings surfaces for database target, Prompt UI, secret vault, and platform-managed LLMs.
